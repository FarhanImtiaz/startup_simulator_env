# MASS Startup Simulator Space

This Space demonstrates MASS, a multi-agent startup decision environment.

Use the **Live Episode** tab to run a simulated startup episode. Tech, Growth,
and Finance co-founders propose actions, and the CEO chooses a final action.

Use the **Training Result** tab to view the CEO SFT loss curve and before/after
metrics from the trained Qwen LoRA CEO policy.

The OpenEnv manifest is `openenv.yaml`.
